# Book Listing
 This is an Android application built with Java, which uses Google Books API to display the books searched by the user. 
 Initially, the app looks like the following image:
<br>
<p align="center">
    <img src="https://user-images.githubusercontent.com/46050303/104578160-64b02480-5680-11eb-803f-24a23b16ed6c.jpg" alt="Home" height="30%" width="30%"></p>
<br>
<br>
Now, the user has to search about a book.
<br>
<br>
<p align="center">
    <img src="https://user-images.githubusercontent.com/46050303/104578158-62e66100-5680-11eb-8aa8-acf3a430a6cf.jpg" alt="Search" height="30%" width="30%">
    <img src="https://user-images.githubusercontent.com/46050303/104578174-67127e80-5680-11eb-81cc-395a180034f1.jpg" alt="Search" height="30%" width="30%">
</p>
<br>
<br>
Then, the search results appear as a list of books.
<br>
<br>
<p align="center">
    <img src="https://user-images.githubusercontent.com/46050303/104578173-6679e800-5680-11eb-8487-1a581e8a2b2c.jpg" alt="Search Results" height="30%" width="30%"></p>
<br>
<br>
On clicking any of the list items, a page with further details about that particular book opens in the browser.
<br>
<br>
<p align="center">
    <img src="https://user-images.githubusercontent.com/46050303/104578170-65e15180-5680-11eb-9420-cff94e43c1fa.jpg" alt="Search Results" height="30%" width="30%"></p>
<br>
<br>
If no relevant results are found, a text appears on the screen saying "No books found".
<br>
<br>
<p align="center">
    <img src="https://user-images.githubusercontent.com/46050303/104578166-6548bb00-5680-11eb-9255-009f57c9c98e.jpg" alt="Search Results" height="30%" width="30%"></p>
<br>
<br>
If there is no internet connection, a text appears on the screen saying "No internet connection".
